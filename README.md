# Simple ML Chatbot

An end-to-end intent-based chatbot built with Python, scikit-learn, and Streamlit.

## How it Works
The bot uses Natural Language Processing (TF-IDF vectorization) to process user text and a Logistic Regression model to classify the text into predefined intents (like greetings, goodbyes, and college info).
